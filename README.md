# SIMPLE TODO APP

## INTRODUCTION

This project implements a simple todo app that integrate [AppRun](https://github.com/yysun/apprun) as fron end in a Spring Boot project.

## RUN THE PROJECT

Simple run the gradle task `bootRun`:
```
gradlew bootRun
```
